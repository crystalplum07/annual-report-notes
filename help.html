📌 系统简介

本系统用于记录和管理年报客户分享会议的纪要，包括客户问答、待跟进事项等信息。

系统支持以下核心功能：

  • 📝 录入客户分享会议纪要
  • 👥 管理用户账号和权限
  • ⚙️ 自定义配置公司、材料、提问方等字段
  • 📊 查看待跟进事项汇总
  • 📥 Excel 批量录入和导出
⚠️ 注意：本系统部署在公开网络上，但只有登录用户才能录入和查看数据。请确保不要录入敏感信息。

👤 所有用户适用

1. 注册与登录

注册新账号：
  1. 在任意页面点击右上角 "登录"
  2. 在登录框中点击 "立即注册"
  3. 输入用户名（2-20个字符）和密码（至少6位）
  4. 点击 "注册"，注册成功后会自动登录
登录：
  1. 点击右上角 "登录"
  2. 输入用户名和密码
  3. 点击 "登录"
💡 提示：注册时系统会自动生成一个内部邮箱地址，无需验证，注册后即可使用。

2. 录入纪要

操作步骤：
  1. 登录后，点击顶部导航的 "录入纪要"
  2. 填写基本信息：
    • 公司名称：从下拉框选择，或输入新公司名称
    • 分享日期：点击输入框选择日期
    • 分享材料：从下拉框选择材料类型
    • 分享对象：输入参会客户人员，如"xx人寿CFO、总精，及精算部员工"
    • KP 负责人：输入负责本次分享的 KP 员工姓名
  3. 填写客户问答（可添加多个）：
    • 点击 "+ 添加问答" 添加更多问答块
    • 每个问答块包含：材料页码、KP问题、客户回复、客户问题、KP回复
    • 点击问答块右上角的 "删除" 可删除该问答
  4. 填写 KP 待跟进事项（可添加多个）：
    • 点击 "+ 添加待跟进" 添加更多待跟进事项
    • 每个待跟进事项包含：材料页码、待跟进内容、跟进状态
    • 跟进状态可选择"进行中"或"已完成"
  5. 点击 "保存纪要" 提交
  6. 保存成功后会出现确认弹窗，点击"确定"可继续录入下一条
📝 注意：带 * 的为必填项，必须填写才能保存。

3. 查看和编辑自己的纪要

管理员用户：
  1. 点击顶部导航的 "纪要管理"
  2. 在纪要列表中找到自己的记录（可通过筛选条件快速查找）
  3. 点击操作列的 "查看" 查看详情
  4. 点击 "编辑" 修改纪要内容
  5. 点击 "删除" 删除纪要（不可恢复）
普通用户：
  1. 目前普通用户需要通过管理员帮助才能查看和编辑自己的纪要
  2. 管理员正在开发普通用户的"我的纪要"页面

4. Excel 批量录入

操作步骤：
  1. 在"录入纪要"页面，滚动到最下方的 "Excel 批量录入" 区域
  2. 点击 "下载 Excel 模板" 下载标准模板文件
  3. 按模板格式填写数据（注意：不要修改表头行）
  4. 填写完成后，点击 "上传 Excel 批量录入"
  5. 选择填写好的 Excel 文件
  6. 系统会自动解析并批量录入，完成后提示录入成功条数
📝 模板格式说明：
列名 说明 是否必填
公司名称 客户公司名称 必填
分享日期 格式：YYYY/MM/DD 必填
分享材料 材料类型名称 必填
分享对象 参会客户人员 必填
KP负责人 KP 负责人姓名 选填
材料页码 问答相关的材料页码 选填
KP问题 KP 提出的问题 选填
客户回复 客户的回复内容 选填
客户问题 客户提出的问题 选填
KP回复 KP 的回复内容 选填
待跟进事项 待跟进的内容 选填
跟进状态 进行中 / 已完成 选填
⚠️ 注意：批量录入时，如果公司名称、分享材料等在配置中不存在，系统会自动添加到配置中。

🔑 管理员专属功能

管理员 以下功能仅管理员账号可见和可操作。

1. 纪要管理

查看纪要列表：
  1. 点击顶部导航的 "纪要管理"
  2. 页面显示所有用户录入的纪要列表
  3. 可查看统计卡片：总计客户数、已分享客户数、已分享占比、待跟进中数量
筛选纪要：
  1. 按公司筛选：选择下拉框中的公司名称
  2. 按材料名称筛选：选择下拉框中的材料类型
  3. 按页码筛选：输入材料页码（如：P5）
  4. 搜索：输入关键词（公司、内容、人员等）
  5. 点击 "搜索" 或按回车执行筛选
查看纪要详情：
  1. 点击某条纪要操作列的 "查看"
  2. 弹窗显示完整的纪要信息，包括基本信息和所有问答、待跟进事项
  3. 点击弹窗右上角的 "×" 关闭
编辑纪要：
  1. 点击某条纪要操作列的 "编辑"
  2. 页面显示编辑表单，可修改所有字段
  3. 修改完成后点击 "保存修改"
  4. 点击 "取消" 放弃修改
删除纪要：
  1. 点击某条纪要操作列的 "删除"
  2. 系统会要求确认，点击 "确定" 永久删除
  3. 删除后无法恢复，请谨慎操作！

2. 用户管理

访问用户管理：
  1. 点击顶部导航的 "信息配置"（用户管理在页面下方）
  2. 页面显示用户统计：总用户数、管理员数、普通用户数
  3. 下方显示所有用户的列表
修改用户角色：
  1. 找到目标用户，点击操作列的 "改角色"
  2. 选择新角色（普通用户 / 管理员）
  3. 点击 "确认修改"
  4. 不能修改自己的角色
重置用户密码：
  1. 找到目标用户，点击操作列的 "重置密码"
  2. 输入新密码（至少4位）
  3. 再次输入确认新密码
  4. 点击 "确认重置"
删除用户：
  1. 找到目标用户，点击操作列的 "删除"
  2. 系统会要求确认，点击 "确定" 删除
  3. 删除用户会同时删除该用户的所有纪要，且无法恢复！
  4. 不能删除自己的账号

3. 信息配置

访问信息配置：
  1. 点击顶部导航的 "信息配置"
  2. 页面显示所有可配置项
配置公司名称：
  1. 在"公司名称配置"文本框中，每行输入一个公司名称
  2. 这些公司会出现在"录入纪要"页面的"公司名称"下拉框中
配置分享材料：
  1. 在"分享材料配置"文本框中，每行输入一个材料类型
  2. 这些材料类型会出现在"录入纪要"页面的"分享材料"下拉框中
配置提问方：
  1. 在"提问方配置"文本框中，每行输入一个提问方标签
  2. 这些标签会用于录入页面的问答区块标签
配置跟进状态：
  1. 在"跟进状态配置"文本框中，每行输入一个状态
  2. 这些状态会出现在"待跟进"的"跟进状态"下拉框中
配置录入默认值：
  1. 分享日期默认偏移：0 = 今天，-1 = 昨天，1 = 明天
  2. KP负责人默认值：每次录入时自动填充的负责人姓名
  3. 跟进状态默认值：每次添加待跟进时的默认状态
保存配置：
  1. 完成所有配置后，点击页面底部的 "保存配置"
  2. 系统提示"保存成功"后，新配置将在下次加载页面时生效
  3. 如需恢复默认配置，点击 "恢复默认"

4. 数据导出

导出待跟进事项：
  1. 在"纪要管理"页面，滚动到"待跟进事项汇总"区域
  2. 点击 "导出 Excel"
  3. 系统生成 Excel 文件并自动下载
  4. Excel 包含：序号、材料名称、材料页码、提问公司、KP负责人、待跟进事项、目前进度、跟进状态
导出客户信息：
  1. 在"纪要管理"页面，滚动到"客户信息汇总"区域
  2. 点击 "导出 Excel"
  3. 系统生成 Excel 文件并自动下载
  4. Excel 包含：序号、材料名称、材料页码、提问公司、KP负责人、KP问题、客户回复、客户问题、KP回复
💡 提示：导出的 Excel 文件格式可以直接打印或转发给团队成员。

❓ 常见问题

Q1：忘记密码怎么办？

请联系系统管理员，管理员可以在"信息配置"页面的"用户管理"区域为您重置密码。

Q2：为什么我无法看到"纪要管理"和"信息配置"菜单？

这两个菜单只有管理员账号才能看到。如果您需要管理员权限，请联系系统管理员为您提升权限。

Q3：录入纪要时，下拉框中没有我想要的公司/材料怎么办？

您可以直接在输入框中输入新的公司名称或材料名称，系统会自动保存并在下次显示在下拉框中（需要管理员在"信息配置"中保存配置后才会正式生效）。

Q4：批量上传 Excel 时提示错误怎么办？

请检查：1) 是否使用了最新的模板；2) 必填列是否已填写；3) 日期格式是否为 YYYY/MM/DD；4) 文件是否为 .xlsx 或 .xls 格式。

Q5：删除的纪要可以恢复吗？

不可以。删除纪要是永久性的，无法恢复。请在删除前仔细确认。

Q6：为什么我注册时提示"email rate limit exceeded"？

这是因为 Supabase 邮件发送频率限制。您可以：1) 等待几分钟后重试；2) 联系管理员在 Supabase Dashboard 手动创建账号。

Q7：系统数据存储在哪里？安全吗？

所有数据存储在任何云 Supabase 数据库中。虽然网站部署在公开网络上，但只有登录用户才能访问数据。请不要在纪要中录入高度敏感的信息。
